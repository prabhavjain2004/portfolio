# Prabhav Jain Portfolio

AI-powered portfolio website built with Next.js 15, React 19, and Tailwind CSS.

## Features

- Interactive AI chat interface
- Traditional portfolio view
- Responsive design
- Modern animations with Framer Motion

## Tech Stack

- **Framework**: Next.js 15.5.7
- **Frontend**: React 19.2.1, TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Deployment**: Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# or
npm run vercel-build
```

## Deployment

This project is configured for deployment on Vercel with automatic builds from the main branch.
